# Prisma (backend)

Kökten workspace script’leri:

- `npm run prisma:generate` — client üret
- `npm run prisma:migrate` — geliştirme migrasyonu
- `npm run prisma:push` — şemayı DB’ye it (prototip)
- `npm run prisma:studio` — Studio

Şema dosyası: `backend/prisma/schema.prisma`. `DATABASE_URL` backend sürecinde tanımlı olmalı.
